Student Result Calculator

This project was developed using Claude AI, HTML, CSS, and JavaScript.

Features:
- Takes marks of 5 subjects
- Calculates Total Marks
- Calculates Percentage
- Displays Grade (A, B, C, D, F)
- Shows Pass/Fail Status
- Validates marks between 0 and 100

Developed as part of AI-Assisted Web Application Development Using Claude.
